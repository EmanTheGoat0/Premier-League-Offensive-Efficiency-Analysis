Premier League Offensive Efficiency Analysis

Project Overview

This project is an automated football analytics pipeline that simulates Premier League performance data to visualize offensive efficiency. By analyzing the relationship between Actual Goals Scored and Expected Goals (xG), this tool identifies which teams are "clinical finishers" (outperforming their xG) versus those who are "wasteful" (underperforming their xG).

Key Features

Data Pipeline: Developed a Python pipeline to simulate realistic Premier League datasets, bypassing the need for fragile web scraping dependencies.

Feature Engineering: Calculated an xG_Difference metric to quantify finishing quality—a key performance indicator (KPI) used by professional football analysts.

Data Visualization: Created a high-fidelity scatter plot using Seaborn and Matplotlib to visualize the relationship between creative output and actual conversion.

Business Insight: Identified team performance outliers, providing a clear visual representation of offensive efficiency across the league.

Data Visualization

The chart below illustrates the offensive efficiency for the 20 teams. Teams positioned significantly above the red line (e.g., Aston Villa) are clinical finishers, while those below are currently underperforming their creative metrics.

Technologies Used

Python: Core programming language.

Pandas: Data manipulation and statistical calculations.
